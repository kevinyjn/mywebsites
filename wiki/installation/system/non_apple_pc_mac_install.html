Non-Apple PC Install Mac OS X

Install Mac Prepare No.1: Get DSDT.aml

提取DSDT.dsl，然后用ACPI Patcher 的 D/C DSDT 功能编译成dsdt.aml

  • 方法1：从Everest程序中提取DSDT，在Everest程序的底部状态栏右击选择提取DSDT，生成acpi_dsdt.bin，然后用ACPI Patcher制作成DSDT.dsl
  • 方法2：Patcher02Beta5 工具下有IASL程序，在cmd命令下直接 iasl -g 提取DSDT.dsl

Install Mac Prepare No.2: /Extra/Extentions Drivers

  • /Extra/Extensions 下必须有 FakeSMC.kext 驱动
  • 最好加上 ElliottForceLegacyRTC.kext，防止刷BIOS
  • NullCPUPowerManagement.kext
  • AppleACPIPS2Nub.kext ApplePS2Controller.kext

Install Mac Prepare No.3: /Extra/org.chameleon.Boot.plist

Install Mac Prepare No.4: /Extra/dsdt.aml

Install Mac Prepare No.5: /Extra/smbios.plist

Install Mac

安装过程中最好加-v -f 参数，根据Reference中新手常见(五国)(-v图)错误解决 解决安装中遇到的错误。
